# Frontend

Vue 3 SPA для тестового задания Laravel + Vue.

## Запуск

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

По умолчанию frontend ожидает backend по адресу:

```env
VITE_API_BASE_URL=http://localhost:8000
```
